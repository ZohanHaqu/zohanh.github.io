About Zohan Haque

Zohan Haque is a software developer who creates simple yet useful tools. His projects include text editors and shells designed to improve everyday tasks for developers.

Some of his creations include Viso Shell, a simple shell for interacting with the system, and Scracky, a text editor meant for basic text editing and file management.
